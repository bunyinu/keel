How it works

  1. Create an account, add a project from the dashboard.
  2. npm install -g @keel2026/cli
  3. keel onboard "your task" --accept "tests pass"
  4. Optional: keel cloud link to sync across machines.
  5. Run your agent. State survives compaction.
