How it works

  1. Create an account, then add projects from your dashboard.
  2. Install CLI: npm install -g @keel2026/cli
  3. In your repo: keel onboard "your task" --accept "tests pass"
  4. Optional: keel cloud link to sync across machines.
  5. Use your agent — state survives compaction.
